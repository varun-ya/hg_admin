# HomeGuru Admin Dashboard

Admin panel for the HomeGuru platform — built with Next.js 15, TypeScript, Tailwind CSS, and Recharts.

## Modules

- **Dashboard** — Global Ops Pulse, Financial Reconciliation, AI Media Health, Governance Queue, Live Classroom Radar, Audit Trail
- **Students** — KPI Ribbon, Master Data Table, 360° Profile Drawer
- **Teachers** — KPI Ribbon, Master Data Table, 360° Profile Drawer
- **KYC Pipeline** — Triage Queue, Full-screen Verification Workspace
- **Agents/Staff** — Staff Directory, RBAC Matrix, Staff Audit Drawer
- **Live Ops** — Class Monitor, Content Moderation, Dispute Resolution, Lead Management
- **Financial Oversight** — Revenue & Escrow, Payouts, Refunds, Invoices & Taxes
- **AI Governance** — Usage Metrics, Quotas & API Keys, Workflow Rules, Ethics Controls
- **System & Security** — Audit Logs, Feature Flags, Emergency Kill Switches, Overrides

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000/dashboard/admin](http://localhost:3000/dashboard/admin)

## Tech Stack

- Next.js 15, TypeScript, Tailwind CSS, Recharts, Phosphor Icons, Lucide React
